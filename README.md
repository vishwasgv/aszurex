# AszureX Website

Professional support company website built with Node.js and Express.

## Quick Start

1. Install dependencies:
   ```
   npm install
   ```

2. Start server:
   ```
   npm start
   ```

3. Open browser:
   ```
   http://localhost:3000
   ```

## Customization

Edit `public/js/config.js` to change:
- Company info
- Services
- Contact details
- Job openings

## Email Setup

1. Use Gmail with App Password
2. Update credentials in server.js (lines 40-41)
3. Enable 2FA: https://myaccount.google.com/security
4. Create App Password: https://myaccount.google.com/apppasswords
